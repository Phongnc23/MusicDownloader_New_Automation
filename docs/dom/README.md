# DOM Reference

Folder chua DOM mau cua tung man hinh app (luu duoi dang markdown).

## Cach thu thap DOM

1. Mo Appium Inspector ket noi voi app
2. Navigate den man hinh can capture
3. Click **Refresh** (icon mui ten xoay)
4. Click tab **Source** -> copy XML
5. Luu vao file `<ten_man_hinh>_dom.md` trong folder nay

## File du kien

- `home_dom.md`        - man hinh Home
- `search_dom.md`      - man hinh Search Online (download)
- `drawer_dom.md`      - Menu / Drawer
- `tracks_dom.md`      - tab Tracks
- `artists_dom.md`     - tab Artists
- `albums_dom.md`      - tab Albums
- `playlists_dom.md`   - tab Playlists
- `search_lib_dom.md`  - Search trong thu vien
